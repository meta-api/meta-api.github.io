Use a Spell's variable in a Connector

Connect a variable​

In the configuration of your connector, you can add variables that can be required or optional. You can fill in the fields directly or pass them variables that you have defined in your code editor. When you select a field, a dropdown menu showing the different variables to which you have access.

Remove a variable​

To delete a variable configured in your connector, you have a cross next to your variable.

Array/Object Specification​

In some cases the variable requested may be an array or an object, in which case it is impossible to define it directly in the field. In this case, the data must be passed as a variable in the editor and then integrated into the field as seen above.
