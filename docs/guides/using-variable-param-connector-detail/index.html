Using variable as parameters in Connector

Setting variables into differents parameters#

You can setting variables as your parameter in your Connector's configuration. This will give you the possibility to make your spell dynamic and to connect different Connectors between them.

Variables are available for the parameters in Query, Header, Form. You can use variable in Server/URL query and in OAuth 2 authentication to select any authentication profiles.

To do so and in first place, you will have to create a variable into your code editor:

const my_variable = "some text";

Then, select one of your Connector on the menu on the left and look for a parameter.

You will notice a button on the right of the input . Hovering it will open a tooltip with a select box.

Here you can search for any variable written in your code. Let's select our variable my_variable. The tooltip should close itself, if it's not the case: just click outside the box.

Now, the input parameter will change to look like this:

There is 2 things:

  • The input parameter is now read-only with the variable set;
  • The button changes its color to a blue. This indicates your variable is now set.

And you're done ‚úÖ

Your parameters, servers and profiles are now set dynamically with a variable.

Remove variable of a parameter#

To unselect a variable, open the tooltip by clicking on the blue button then click on the unselect variable button. This will remove any reference of this variable in the parameter.
