Code and Dependancies

Code#

Write your Javascript code directly into the code editor.

You can create as much as function you need or want inside the editor.

You will find all the parameters into the params variable. We recommend you to deconstruct it like this: const { param1, param2 } = params

Dependencies#

You can add NPM dependencies from Manage dependencies menu on the right: search the package, give it an alias (or not) and save! It will be added on deployment.

An alias is the name for the package inside your code. Call the library by this name.
