Use your Spell in production

Release and deploy to prod

Before deploying in producton, you will need to crete a new release of your code from the one you have in development. For this, go on release tab and click Create a new release.

Your development code will be automatically versioned.

Then click on "Promote to production* to deploy this release on your production environment.

How to use your Spell in your project

To use your Spell, you can do either a GETor a POST (if you need to send data) on the production URL that you'll find on release tab. You can also find some snippets to help you integrate the Spell.

Don't forget to add your API key (on your client account) either into the request's headers : apikey: YOUR_API_KEY or into the parameters of the URL : https://api.meta-api.io/api/spells/5ddbe9c2675f0f0012afb414/runSync?apikey=YOUR_API_KEY
