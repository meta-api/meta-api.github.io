Connectors' authentication

Add and configure authentication of a connector#

Many API need an API key or an authentication to send requests. You can configure the authentication in your client account, that way you'll manage your API keys more easily.

In that purpose, go to your client account and under Auth & Connectors you will be able to add any connector you need.

When the connector is added, select it and add the API key or anything needed for the authentication.

Manage connectors#

You can access a the configured connectors list at any moment to update it whenever you want.
