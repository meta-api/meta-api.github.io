Deploy and test your Spell

During Development :#

Your Spell is deployed when you save it. Click on Save and that's it! You can now test your code.

Tester :#

To test, click the Test your Spell button: a panel will allow you send a request in order to test your deployed code and check the results.

Deployment's errors#

  1. To accelerate your development, your code is deployed temporary. If the development environment doesn't respond (it turns off if nothing happens during 20 minutes), restart it by clicking on Reset the dev environment under the Information tab.
  2. Then, you will need to save your code in order to redeploy it.
