Meta Storage, a key-value storage directly inside the Spell

Description and examples‚Äã

Meta Storage is a key/value database integrated inside your Spells. You can set, retrieve and share values between Spells and executions.

It's a great way to share data between Spells or to synchronize and keep a state between each executions.

For example, we love when we are synchronizing two services, like Pipedrive (a CRM) and Airtable (a spreadsheet). With Meta Storage, we can store the last synchronization date and the last ID to limit requests done to the services.

How to use it‚Äã

Set a key‚Äã

You can set a new key and value using the following syntax:

await metaStorage.set(key, value)

Example to set the value "harry" to key "wizard":

await metaStorage.set("wizard", "harry")

You can store value, object or array inside any key.

Get a key‚Äã

You can retrieve a value associated to a key using the following syntax:

await metaStorage.get(key)

Example to get the value of key "wizard":

const myValue = await metaStorage.get("wizard") //Will contain "harry"

Delete a key‚Äã

You can delete a key using the following syntax:

await metaStorage.delete(key)

Example to delete the key "wizard":

await metaStorage.delete("wizard")
