Advanced data manipulation and snippets

Work with files and images#

With our code editor, you can manage files and images. Either as a body from the HTTP call, as a resource you are fetching in your code or as a return result.

You have two ways for managing files and images inside a Spell : Buffer or local files.

Managing with buffer#

With a Buffer, you can store files directly inside the Spell's memory.

Here a snippet to store a remote file inside a Buffer : 

const imageInBuffer: Buffer = await axios.get("https://images.unsplash.com/photo-1546776310-eef45dd6d63c?ixid=MXwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHw%3D&auto=format&fit=crop&w=739&q=80", {    responseType: 'arraybuffer'});

You can also create a Buffer from a string (utf8) or an object : 

const fileInBuffer = new Buffer.from(myObject);

Or a base64 string :

const fileFromBase64 = new Buffer(base64String, 'base64');

Then you can manipulate this variable to output a file (see Generate a file from a Spell) or to link it inside a POST request, etc.

Warning : memory are limited inside a Spell. If you tried to store too large files inside a Buffer, you'll break your Spell's execution.

Read and save temporary files#

You'll need to activate the fs Node's library to access Spell's filesystem.

Then you can read and write files from the local tmp directory like this:

//Write a new file named "test" inside the tmp directory and putting inside the content "Hello World"fs.writeFileSync('test', "Hello World");
//Loading the file's content inside a Bufferconst fileContentBuffer = fs.readFileSync('test');
//Convertir Buffer to string to use the contentconst fileContentAsString = fileContentBuffer.toString();

Output a file#

You can output a file from a Spell to interact direcly with user and propose them to download it.

You have a special object _file that you can return from a Spell : 

return {    _file: {        filename: "test.csv" //You can choose here the filename of your file. This filename will be proposed to the user when he will download it        data: new Buffer.from(csv) //data must be a Buffer    }}

Here a full example of a Spell with output a CSV : (we are using json2csv parser lib to convert a JSON to a CSV)

const someData = [    {        "Firstname": "John",        "Lastname": "Doe",        "Language": "English",        "Age": 31    },    {        "Firstname": "Alain",        "Lastname": "Terrieur",        "Language": "French",        "Age": 32    },]
const json2csvParser = new json2csv.Parser();const csv = json2csvParser.parse(someData);
return {    _file: {        filename: "people.csv",        data: new Buffer.from(csv)    }};

Here the output inside the people.csv file :

"Firstname","Lastname","Language","Age""John","Doe","English",31"Alain","Terrieur","French",32
