Testing Spells

Once your spell is ready, you can test it with our development environment available for 60 minutes after you clicked on Run button

When you are satisfied with your spell, you can create a new version and push it to production ! ðŸŽ‰

To test it, you can use your favorite web tool for HTTP requests or our editor directly.

Test with Meta-API#

If you need to send data to your spell, click on the Payload button and fill the JSON Once you click on the Run button, the Console & Response will open and load your spell instantly.

From there you can see the formatted JSON response of the HTTP call and the output in the console if you added some logs.

Test with Curl#

From your terminal, you can access our development environment by adding dev=true to your spell url.
