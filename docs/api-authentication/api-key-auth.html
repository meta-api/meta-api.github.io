Use API Key

API key authentication is one of the simplest way to authenticate on an API.

Go to your provider website and look for your API Key. You can find it in your settings most of the time.

Then, you'll have to paste your API key inside the authentication popup.

caution

Check for spaces before and after your API key that can prevent it to work
