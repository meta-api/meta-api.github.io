How to release a Spell in production?

Once your spell is ready to go live, you can create a production release from the Release tab.

On the top you can see your current version in development. To create a new release, you can click on the button Draft a new release which will prompt you to set a title and a description for your release.

Now that you have created your release, you can promote it to production when you are ready.
