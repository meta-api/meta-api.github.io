Use a Spell in production

Once your Spell has been deployed to Production you can call it with a simple HTTP request.

We display for you in the release tab of you spell a list of snippets to test immediately along with the production url.

Select your Spell key from the dropdown menu to fill automatically the header apiKey in the snippets.

note

Your development URL is only up for 15 minutes and must be used for testing only
