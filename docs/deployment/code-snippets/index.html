Use a Spell in production

Once your Spell has been deployed to Production you can call it with a simple HTTP request.

We display for you in the release tab of your spell a list of snippets to test immediately along with the production url.

Select your Spell key from the dropdown menu to fill automatically the header apiKey in the snippets.

note

Your development URL is only up for 15 minutes and must be used for testing only

Your spell is accessible via GET and POST requests, use the most appropriate for your use case. A sample for each method is available at the bottom of the Snippets section. All responses are in the json format.

important

To make an HTTP request to Meta-API you need to provide an API key in the header with the key apikey
