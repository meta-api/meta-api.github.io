API documentation and how to consult it

For every API a documentation is available from the API description in the Documentation tab.

Your can download the OAS (Open API Specification) of the API by clicking on the Download button.

All of the available endpoints for the API are listed on the left side of the screen with a sample of the corresponding request on the right.

Our generated documentation respect the Open API format in it latest version 3.1.0.

To provide the best documentation possible, please make sure to provide the following for each endpoints :

  1. Summary
  2. Description
  3. Required and optional fields
  4. Sample response for a 2XX
  5. Sample response for a 4XX
