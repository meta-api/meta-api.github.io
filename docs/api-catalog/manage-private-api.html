Integrate and manage a private API

We provide more than 100 APIs to connect with your spells, however you may also need to use your own private APIs.

Good news! You can add as much as you want in a few easy steps.

Add an API​

In the left sidebar of the dashboard you can add your API by clicking on Connectors and press the button Create your connector in the top right corner.

You must fill the following required fields :

  • Title : 3 words or fewer to recognize your API
  • Provider : domain like meta-api.io
  • Description : short sentences summarizing the purpose of your API

Then press the save button in the top right corner.

Once saved, you can upload your YAML or JSON file in the Open API format version 2 or 3.

If your API is not available in the OAS format, you can do it with a tool like Stoplight

For the best use cases and if you plan to make your API public, we advise you to respect OAS conventions in your specification to provide the best documentation possible.

Update your API​

From the list of connectors, you can search for your own and click on it to update it.

All fields and the OAS file can be updated at any time.

You can also upload a new JSON or YAML file to update the whole connector

Remove your API​

To delete your API, you just need to click on the Delete button available in the API settings.
