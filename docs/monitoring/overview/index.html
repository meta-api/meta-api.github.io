Monitoring of your Spells

Home pageâ€‹

You'll find an overview of all your spell running in production on the home page. That's the easiest way to check if everything's good or if your attention is needed.

On the left, you'll see the production logs from the last 48 hours. If you need more informations, visit the logs page by clicking on Go to your monitoring...

On the right, the 3 charts will show you the number of errors, successful calls and average exection time. You can select the time period from the last 48 hours to the last year.

Spell's detailsâ€‹

On the Spell's detail page, you'll find the same informations than on the home page but only for the selected Spell.

The yellow line on the charts represent the last deployment in production. It will help you compare the impact of the new version from the previous ones.

Statisticsâ€‹

The statistics page will show you the number of calls and errors for each day of the selected month, the average execution time and the number of active Spells.

Logsâ€‹

The logs page will show you all the logs from your Spells. You can filter the Spell, the date range, the environment and the errors only.

You can open the details od one specific log by clicking on the arrow on the right. You'll see the error message, the console informations (the console.log from your code) and the connectors list with there execution time and OAS validation from the request and the response.

note

The Mail limitation checkbox prevent you from receiving more than 5 error notifications emails per hour for the same Spell
